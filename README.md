# BudgetFlow

BudgetFlow is a modern offline-first personal budget tracker built with React, Zustand, Tailwind CSS, and Vite.

Designed to help users manage daily allowance budgets using a calendar-based tracking system.

## Features

- Daily budget tracking
- Quick accumulated withdraw system
- Monthly navigation
- Calendar history restriction
- Analytics dashboard
- Configurable daily budget
- Settings modal
- Backup / Restore JSON
- Responsive UI
- Smooth animations
- Offline support (PWA)
- LocalStorage persistence

## Tech Stack

- React
- Vite
- Zustand
- Tailwind CSS
- Framer Motion
- Lucide React
- PWA
